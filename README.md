# 2025_ZhiHuiShu
智慧树(知到)自动刷课禁音脚本

&gt; 基于 [2024智慧树知自动播放](https://greasyfork.org/zh-CN/scripts/494796-2024%E6%99%BA%E6%85%A7%E6%A0%91%E7%9F%A5%E8%87%AA%E5%8A%A8%E6%92%AD%E6%94%BE) 的修改版本，调整了完播判定、下一节跳转与静音逻辑，并支持自定义倍速。

---

## 📌 脚本功能

- ✅ 完播逻辑修改：视频播放到 **99.9%** 即视为完播
- ✅ 跳转逻辑修改：`field + 1`
- ✅ 静音逻辑修改
- ✅ 支持自定义播放倍速（默认 1.0x）
- ✅ 完播后自动播放下一节，无需手动点击
